# GJ Camp Website - Frontend

Frontend React pour le site de la jeunesse de l'église.

## Installation

```bash
npm install
```

## Démarrage en mode développement

```bash
npm start
```

Le site sera accessible sur `http://localhost:3000`

## Structure

- `src/components/` - Composants réutilisables (Header, Footer, Carousel)
- `src/pages/` - Pages principales (HomePage, LoginPage, SignupPage)
- `src/context/` - Contexte d'authentification
- `src/styles/` - Fichiers CSS
